📋 Quy Chế Thi

Thời gian thi: 3 giờ (180 phút)

Số lượng bài tập: 3 bài tập lập trình

Ngôn ngữ được phép: C, C++, Python, Java

Cách chấm điểm: Dựa trên độ khó và thời gian hoàn thành

Quy tắc nộp bài: Chỉ được nộp file code (.c, .cpp, .py, .java)

🔨 Luật Thi

  • ✔️ Thí sinh phải hoàn thành bài thi trong thời gian quy định
  • ✔️ Không được sử dụng tài liệu bên ngoài hoặc internet
  • ✔️ Mỗi bài chỉ được nộp tối đa 10 lần
  • ✔️ Thí sinh có thể xem kết quả chấm ngay sau khi nộp
  • ❌ Nghiêm cấm gian lận hoặc sao chép code
  • ❌ Không được rời khỏi phòng thi khi chưa hoàn thành

Lưu Ý Quan Trọng

  • • Đảm bảo kết nối internet ổn định trong suốt quá trình thi
  • • Kiểm tra định dạng file code trước khi nộp bài
  • • Thời gian sẽ được tính từ khi bấm nút "Bắt đầu làm bài"
  • • Liên hệ giám thị nếu gặp vấn đề kỹ thuật
